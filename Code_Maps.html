Key Components

  • src/app.py: The "Main Loop". It initializes the database, config, and renders the Streamlit UI.
  • src/automation_engine.py: The heartbeat of autonomous missions.
  • src/database.py: The central source of truth.

üíæ Data Flow & Persistence

  1. Input: User uploads CSV or Agent scrapes Web.
  2. Processing: pandas helps clean data; Agents enrich it.
  3. Storage: leads.db (SQLite) stores everything.
  4. Output: CSV Exports or Dashboard Visualization.
Debugging Tip: If data seems missing, check leads.db directly or look at src/database.py to see if the commit() function is being called!
