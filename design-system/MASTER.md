# SMarketer Pro Design System (MASTER)

Inspiration: [ui-ux-pro-max-skill-main](file:///C:/sandbox/b2b_outreach_tool/ui-ux-pro-max-skill-main)

## Core Identity: B2B SaaS Enterprise

Trust-driven, professional, and data-dense.

### Color Palette

- **Primary**: `#0F172A` (Professional Navy) - Used for brand identity, headers, and primary buttons.
- **Secondary**: `#334155` (Slate Grey) - Used for supporting text and neutral states.
- **CTA**: `#0369A1` (Trust Blue) - Used for conversion elements and accents.
- **Background**: `#F8FAFC` (Clean Slate) - Main app background.
- **Text**: `#020617` (Deep Ink) - Primary readability.
- **Border**: `#E2E8F0` (Light Slate) - Subtle separation.

### UI Patterns

- **Style Priority**: Trust & Authority + Minimalism
- **Effects**: Glassmorphism (Backdrop blur 10px, subtle borders, elevated shadows).
- **Transitions**: Smooth 250ms cubic-bezier for interaction feedback.

### Key Components Rules

- **Metrics**: Hover lift effect with trust blue border.
- **Cards**: Glass-like transparency (bg-white/70) with soft shadows.
- **Buttons**: Linear gradient (Navy -> Trust Blue) with depth on hover.

### Anti-Patterns to Avoid

- ❌ Playful neon colors (unless for status alerts).
- ❌ AI purple/pink gradients in enterprise context.
- ❌ Over-animation that slows down data interaction.
- ❌ Missing hover feedback on clickable elements.
