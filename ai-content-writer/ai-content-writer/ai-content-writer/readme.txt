This plugin relies on third-party services to provide its functionality. Below is a detailed explanation of the services used, the data sent, and links to their terms and policies:

* **Purpose**: The Google Gemini API is used to generate content using AI technology.

* **When Data is Sent**: Data is sent to the Google Gemini API when the user generates content using the plugin.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ai-content-writer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to WordPress Admin Dashboard > AI Content Writer > Generate Content to start using the plugin and generate content using AI.
4. Configure your API settings and start generating content.
