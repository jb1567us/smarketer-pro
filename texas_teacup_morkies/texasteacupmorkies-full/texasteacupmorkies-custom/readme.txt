Texas Teacup Morkies Custom Theme

INSTALL
1) Upload this folder (or ZIP) to wp-content/themes/
2) Activate in WP Admin -> Appearance -> Themes
3) Go to Appearance -> TTP Settings and set:
   - Lead Mode (Waitlist or Referral)
   - Waitlist Form URL (Google Form)
   - Phone/Text, Address, etc.
4) Create pages (or assign templates to existing pages):
   - Apply / Waitlist -> Template: Apply / Waitlist
   - Referral Request -> Template: Referral Request
   - Reviews & Happy Homes -> Template: Reviews & Happy Homes
5) Add Puppies:
   - WP Admin -> Puppies -> Add New
   - Set Status to Available/Reserved/Sold in the Puppy Details box
   - Set a Featured Image
   - Add additional photos/videos in the editor content using core blocks.

HOMEPAGE LOGIC
- Homepage automatically features the most recent puppy marked "Available".
- If no available puppies exist, the homepage uses Lead Mode to drive either Waitlist or Referral.

SEO BREED PAGES
- Breed pages are available automatically at:
  /breed/maltipom
  /breed/morkie
- Edit Breeds (WP Admin -> Breeds) and add a Term Description; it appears on the breed page.
