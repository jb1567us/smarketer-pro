Texas Teacup Morkies Custom Theme

- Conversion-focused home page with Featured Puppy card.
- Custom post type: Puppies (Available / Reserved / Sold).
- Lead Mode toggle: Waitlist vs Referral (Appearance → TTP Settings).
- On first activation, the theme creates:
  Home, Apply/Waitlist, Referral Request, Reviews, Process/FAQ, Contact, Maltipom, Morkie pages,

  [ttp_map_directions] — embeds directions from locality center to pickup address.
